# rule34api
api for rule34.xxx
# Install
```
pip install xmltojson
git clone https://github.com/l0v3m0n3y/rule34api
```

### Example

<img src="https://rule34.xxx/favicon.ico" alt="Favicon">

```python3
from rule34api import R34api
from asyncio import run
async def main():
    r34api = R34api()
    await r34api.recomend_posts()
run(main())
```
